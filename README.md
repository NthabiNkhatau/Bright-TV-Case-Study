# Bright-TV-Case-Study
Bright TV analysis
This project analyses BrightTV’s user and viewership data to generate insights that support the company’s goal of growing its subscription base. Using data from user profiles and viewing sessions, the study combines demographic information with content consumption behaviour to understand who the users are, what they watch, and when they engage with the platform. The data was processed and transformed using SQL, where key variables such as age groups, day of the week, time of day, and viewing duration were created. The cleaned and consolidated dataset was then used to develop pivot-style analyses and visualisations in Excel, aligned with key performance indicators such as channel performance, viewing trends, audience segments, and engagement levels.

The analysis reveals clear patterns in user behaviour, including peak viewing periods in the evenings and on weekends, strong engagement driven by sports content, and a dominant user base within the 25–34 age group. It also highlights gaps such as low Monday viewership and underrepresentation of certain audience segments, particularly female users. Based on these findings, the project provides strategic recommendations focused on content scheduling, targeted marketing, and user growth initiatives. Overall, the work demonstrates how data-driven insights can inform decision-making and support BrightTV in increasing user engagement and expanding its subscriber base.
